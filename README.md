## Executando a aplicação Localmente:

1) Instalar os pacotes com o comando:

``` 
> npm install
```

2) Depois executar o seguinte comando para executar a aplicação:

```
> npm run dev
```

Depois bastam abrir o browser em **localhost:8000**
